## Overview

This repository contains categorized problem-solving challenges aimed at strengthening the core thinking skills every programmer needs:

-  Algorithmic thinking

- Logical reasoning

- Basics of hashing

- Breaking problems into steps

- Practicing coding in any language



Each folder includes exercises, examples, and solutions (when applicable).
## Getting-Started


```bash
# Clone the repository
git clone https://github.com/ZeyadGabr1/problem-solving-challenges.git
```

```bash
# Navigate to the project directory
cd problem-solving-challenges

```

Pick any challenge → read → solve!


## Repository Structure

```bash
easy/      # Beginner level challenges
medium/    # Intermediate challenges
hard/      # Advanced problem-solving challenges
```

Each folder contains:

- Clear problem statements

- Optional hints

- Example inputs/outputs

- solutions

## Challenge-Levels

### Easy
- Fundamentals like loops, conditions, strings, simple arrays.

### Medium
- More logic-oriented tasks, patterns, math reasoning, simple algorithms.

### Hard
- Advanced problem-solving, optimized solutions, recursion, DP, tricky logic.
