
# Birthday Cake Candles - HackerRank.com


## Problem Short Description
You are in charge of the cake for a child's birthday. The cake will have one candle for each year of the child's age.  
The child can only blow out the tallest candles.

Your task is to count how many candles have the maximum height.


##  Example

```bash

Input:
candles = [4, 5, 1, 5, 2]

Output:
2
```
